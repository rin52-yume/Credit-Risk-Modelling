This project investigates the impact of various class imbalance correction techniques on the stability of Explainable AI (XAI) feature rankings within credit risk assessment models. Credit default datasets exhibit severe class skew, often necessitating resampling methods like SMOTE or cost-sensitive learning to prevent models from defaulting to majority-class predictions. While advanced ensemble algorithms like XGBoost provide high predictive accuracy, their inherent "black-box" nature requires post-hoc explainers like SHAP (SHapley Additive exPlanations) to ensure regulatory compliance and transparency. A critical challenge arises when imbalance corrections alter underlying feature distributions, potentially yielding unstable explanations where the perceived importance of features shifts drastically. Utilizing the UCI Taiwan Default Credit Card dataset, this study evaluates five imbalance correction methods—SMOTE, ADASYN, Borderline-SMOTE, Class Weights, and Threshold Moving—and quantifies SHAP ranking stability using Spearman Rank Agreement (SRA) across bootstrap samples. The ultimate objective is to identify the optimal balance between predictive performance and explanation reliability, culminating in an interactive Streamlit dashboard designed for transparent, real-time credit decision-making.
